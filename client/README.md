# TaskMaster Frontend (Client)

This is the frontend for the TaskMaster application, built with React, Vite, and Tailwind CSS.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Setup**:
    -   (Optional) Create a `.env` file if you need to point to a production backend.
    -   Default API URL: `http://localhost:5000/api`

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## Build for Production

To create a production build (for Vercel/Netlify):
```bash
npm run build
```
The output will be in the `dist` folder.
